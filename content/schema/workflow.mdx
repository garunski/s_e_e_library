# Workflow schema

Category `workflow`. Payload file `definition.json`. Installs to
`.s_e_e/workflows/definitions/{slug}.json`.

An outer envelope wraps the engine `content`:

- `id` - `system-<name>` (defaults) or `user-<name>`. Required.
- `name`, `description`, `version` - strings.
- `content` - the engine workflow object:
  - `id` - required, non-empty.
  - `name` - required.
  - `tasks[]` - required, non-empty.
  - `runtime_inputs[]` - optional; each `{ key, input_type, required, label }`; `key` must not contain dots.
  - `spoke_root`, `hub_root`, `stories_root`, `knowledge_root` - optional root paths.

## Tasks

Each task:

- `id` - unique within the workflow.
- `name` - string.
- `function` - exactly one handler key: `cli_command`, `command`, `user_input`, `git_action`, `story_action`, `foreach`, or `custom`.
- `next_tasks[]` - optional nested tasks (depth-first execution).

A payload containing `tasks` is treated as a workflow (not a prompt).

## Template expansion

- `{{prompt.ID}}` - injects a stored prompt; the id must exist in the hub.
- `{{runtime.KEY}}` - a run-start value; declared in `runtime_inputs`, no dots.
- `{{userinput.TASK_ID.value}}` - a value from an earlier `user_input` task.

## JSON Schema

Download: [workflow.schema.json](https://garunski.github.io/s_e_e_library/schema/workflow.schema.json)

```json
{
  "$schema": "https://json-schema.org/draft/2020-12/schema",
  "$id": "https://garunski.github.io/s_e_e_library/schema/workflow.schema.json",
  "title": "Workflow package payload (definition.json)",
  "type": "object",
  "required": ["id", "name", "content"],
  "properties": {
    "id": { "type": "string", "minLength": 1 },
    "name": { "type": "string", "minLength": 1 },
    "description": { "type": "string" },
    "version": { "type": "string" },
    "content": {
      "oneOf": [
        { "type": "string", "description": "JSON-encoded workflow object" },
        { "$ref": "#/$defs/workflow" }
      ]
    }
  },
  "$defs": {
    "workflow": {
      "type": "object",
      "required": ["id", "name", "tasks"],
      "properties": {
        "id": { "type": "string", "minLength": 1 },
        "name": { "type": "string", "minLength": 1 },
        "spoke_root": { "type": "string" },
        "hub_root": { "type": "string" },
        "stories_root": { "type": "string" },
        "knowledge_root": { "type": "string" },
        "runtime_inputs": { "type": "array", "items": { "$ref": "#/$defs/runtime_input" } },
        "tasks": { "type": "array", "minItems": 1, "items": { "$ref": "#/$defs/task" } }
      }
    },
    "runtime_input": {
      "type": "object",
      "required": ["key", "input_type"],
      "properties": {
        "key": { "type": "string", "pattern": "^[^.]+$" },
        "input_type": { "type": "string" },
        "required": { "type": "boolean" },
        "label": { "type": "string" }
      }
    },
    "task": {
      "type": "object",
      "required": ["id", "name", "function"],
      "properties": {
        "id": { "type": "string", "minLength": 1 },
        "name": { "type": "string" },
        "function": { "$ref": "#/$defs/function" },
        "next_tasks": { "type": "array", "items": { "$ref": "#/$defs/task" } }
      }
    },
    "function": {
      "type": "object",
      "required": ["name"],
      "properties": {
        "name": {
          "enum": ["cli_command", "command", "user_input", "git_action", "story_action", "foreach", "custom"]
        },
        "input": { "type": "object" }
      }
    }
  }
}
```
