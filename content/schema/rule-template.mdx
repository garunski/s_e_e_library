# Rule &amp; template schema

Two lightweight, file-only categories.

## Rule

Category `rule`. Payload is a Cursor rule `.mdc` file (markdown with rule
frontmatter). Installs to `.cursor/rules/{slug}.mdc`.

## Template

Category `template`. Payload is an arbitrary scaffold file. Installs under
`templates/`.

See the [catalog schema](/schema/) for the manifest and install-path rules.
